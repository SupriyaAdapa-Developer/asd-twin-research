## 1. Hypotheses

**H1 (primary).** Adding environmental covariates (weather, school calendar, staffing, community, family) to a baseline ABA session-features model will improve out-of-sample AUROC by ≥0.03 for the binary outcome of session success.

**H2 (primary).** The environmental covariate block will explain ≥10 additional percentage points of outcome variance (McFadden's pseudo-R² delta) over a Tier 2 model (session features + therapist/schedule features).

**H3 (secondary).** Within the environmental covariate block, the following individual predictors will have FDR-corrected p < .01 and standardized |β| > 0.10 in the direction specified:

- Substitute therapist day → negative effect on session success (specified direction)
- 24-hour barometric pressure delta > ±5 hPa → negative effect (in children with documented sensory sensitivity)
- First/last day of school break → negative effect
- Caregiver-reported family stress event in past 48 h → negative effect

**H4 (exploratory).** The Bayesian per-child hierarchical model will outperform the population-level regularized regression in calibration (Brier) by ≥0.02, while not degrading in discrimination (AUROC).

## 2. Sample

**Synthetic cohort:** n = 184 children, 33,400 sessions, 18-month window. Generated by `data/data_generation.py` with seed = 42.

**Real-data replication (planned):** Two ABA clinics in different US census regions, n ≥ 100 children per site, ≥ 12 months of session data per child. IRB approval at each site, with data use agreement specifying re-identification protections.

**Inclusion criteria.** Children aged 3–17 with documented ASD diagnosis (ICD-10 F84.0), enrolled in active ABA programming for ≥ 3 months at study entry.

**Exclusion criteria.** Children whose sessions are exclusively assessment-based (ADOS, ABLLS-R, VB-MAPP intake assessments) without intervention components. Children for whom < 30% of sessions have any environmental covariate data available.

## 3. Variables

See `docs/methodology.md` for the complete variable definitions, sources, and operationalizations. The pre-registered list of predictors per tier is fixed and will not be added to without filing an amendment to this pre-registration.

**Outcome variables (pre-specified):**

- **Primary:** Session success (binary, ≥ 80% planned trials at criterion)
- **Secondary:** Behavioral incident occurrence (binary, any logged SIB / ATO / property destruction / elopement)
- **Tertiary:** Mastery progression (continuous, change in cumulative mastered targets per goal-week)

**Predictor blocks (pre-specified):**

- Tier 1: 8 baseline session features
- Tier 2: Tier 1 + 7 therapist/schedule features
- Tier 3: Tier 2 + 17 environmental covariates
- Tier 4: Tier 3 + per-child random effects on 4 highest-variance predictors

## 4. Analysis plan

**Software:** Python 3.11.7 with pandas 2.2.0, scikit-learn 1.4.0, statsmodels 0.14.1, pymc 5.10.0, lightgbm 4.3.0. Locked versions in `requirements.lock`. All analysis run inside a versioned Docker image to guarantee bit-identical reproducibility.

**Train/validation/test split.** 70/15/15, stratified by child, with all sessions for a given child in the same partition. Random seed = 42 (synthetic) or = study-site code (real-data replications).

**Primary model class.** Logistic regression with L2 penalty. λ chosen by 5-fold cross-validation on the training set, optimizing log-loss. Coefficients reported with 95% confidence intervals (Wald-type) and standardized effect sizes.

**Secondary model class.** LightGBM with monotonic constraints on a pre-specified subset of predictors (substitute therapist → success: monotonic decreasing; familiar therapist tenure → success: monotonic increasing). Reported as cross-check on linear-model coefficient signs.

**Tertiary model class.** Bayesian hierarchical model in PyMC. Priors on environmental coefficients: Normal(0, 0.5). Priors on per-child random effects: Half-Normal(0.3) for the variance hyperparameter. NUTS sampler, 4 chains, 2000 warmup + 2000 samples, target_accept = 0.95. Convergence: R-hat < 1.01, ESS > 1000 for all parameters.

**Inference rules.**

- A predictor is *reportable as significant* if its FDR-corrected p < .01 (BH method, q = .01) AND its standardized |β| > 0.10.
- A model tier is reported as *superior* to its predecessor if test-set AUROC improves by ≥ 0.03 AND Brier does not deteriorate by > 0.005.
- Posterior predictive checks must pass (Bayesian p-value ∈ [0.05, 0.95]) for the hierarchical model to be reported as the headline finding.

**Decision tree for missing data.**

- Tier 1 predictors: complete by construction. Sessions with missing values dropped.
- Tier 2 predictors: < 5% missingness expected. Multiple imputation by chained equations (m = 20).
- Tier 3 environmental: variable missingness (most missing for caregiver-reported features). MICE with predictor matrix specified in `src/analysis/imputation.py`.
